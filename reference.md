## Software / Services

- [PyCharm](https://www.jetbrains.com/pycharm/) (IDE): part of the [JetBrains](https://www.jetbrains.com/) ecosystem. Free academic licence (1yr) if you sign up with your institutional email address.
- Visual Studio Code ([VSCode](https://code.visualstudio.com/)) (IDE). An MS product but without all the usual garbage.
- [Gitbash](https://gitforwindows.org/) – a git client for windows
- [Anaconda](https://www.anaconda.com/) – a python environment management system.
- [Cookiecutter](https://cookiecutter.readthedocs.io/en/1.7.2/) a project templating system
- [GitHub](https://github.com/), [GitLab](https://about.gitlab.com/), and [Bitbucket](https://bitbucket.org/product) version control repositories
- [BASH](https://www.gnu.org/software/bash/manual/html_node/index.html#SEC_Contents) reference manual
- The python package index ([pypi](https://pypi.org/))
- [Read the docs](https://readthedocs.org/) – a free online documentation hosting service
- [Travis-CI](https://www.travis-ci.com/) – a continuous integration tool
- [Zenodo](https://zenodo.org/) – free and open repository for data, software, papers, and more
- The Astronomy Source Code Library ([ASCL](http://ascl.net/))
- [ORCiD](https://orcid.org/) – a persistent id to connect you to your research outputs
